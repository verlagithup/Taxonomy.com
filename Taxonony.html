TAXONOMY

It is the science of classifiction of living things into taxas or groups on the bases of the features they have in common.

Organisms are classified according to the following kingdoms below.

Kingdom Prokaryotae Kingdom Protoctista Kingdom Plantae Kingdom Fungi Kingdom Animalia

Classification

This is the orderly arrangmemt of living organisms into groups base on the features there have in common

Importance of classification

Biologist classify organisms for the following few reasons below.

  • To eased the study of living organisms.
  • To facilitate identification of organisms.
  • To trace the evolutionary descent of living organisms base on the interelationship existing btw them.
  • To create a scheme so that when a new organism is discovered it can easily be fitted into the scheme.
  • To make the study of biology universal.

Types Of Classifications

  1. Artificial Classification

    2. It places organisms into groups for purpose of "convinience". It classifies organisms base on one or a few observable features and characteristics. And considers only analogous structures

  2. Natural classification

    3. It is base on natural features as well as data gotten from "Anatomy,Physiology,Biochemistry,Ecology" etd... It is base on many features both internaland external. And it is base on "homologous structures"

  3. Nomenclature

    4. It is the giving of names to organisms for identification. Each organism has two names i.e 'Common Name' and 'Scientific Name'.

    • Binomial Nomenclature

      • It is the system of naming living organisms using 'Scientific' names made up of two 'Italic' names. For example humans scientific name is "Homo sapiens". This system is universal.
